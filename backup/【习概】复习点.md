### **第六章：推动高质量发展**
**学习要点：**
1. **新发展理念的科学内涵和实践要求**
2. **高质量发展的深刻内涵和重大意义**
3. **坚持和完善社会主义基本经济制度**
4. **构建以国内大循环为主体、国内国际双循环相互促进的新发展格局**

**小标题：**
* **第一节：完整、准确、全面贯彻新发展理念**
    1. 我国进入新发展阶段
    2. 贯彻新发展理念是关系我国发展全局的一场深刻变革
    3. 以新发展理念引领高质量发展
* **第二节：坚持和完善社会主义基本经济制度**
    1. 坚持和完善社会主义基本经济制度是实现高质量发展的保障
    2. 坚持「两个毫不动摇」
    3. 坚持按劳分配为主体、多种分配方式并存
    4. 构建高水平社会主义市场经济体制
* **第三节：加快构建新发展格局**
    1. 把握未来发展主动权的战略部署
    2. 以国内大循环为主体、国内国际双循环相互促进
    3. 大力推动构建新发展格局
* **第四节：建设现代化经济体系**
    1. 建设现代化产业体系
    2. 全面推进乡村振兴
    3. 促进区域协调发展

其余知识点：

1.  **我国经济发展的历史方位**：我国经济已由**高速增长阶段转向高质量发展阶段**。进入新发展阶段，是实现第一个百年奋斗目标后，向第二个百年奋斗目标进军的新起点。
2.  **新发展理念的辩证关系**：**创新**是第一动力，解决动力问题；**协调**是内生特点，解决不平衡问题；**绿色**是普遍形态，解决人与自然和谐问题；**开放**是必由之路，解决内外联动问题；**共享**是根本目的，解决社会公平正义问题。
3.  **高质量发展的判断标准**：从「有没有」转向「好不好」。它要求实现**投资有回报、企业有利润、员工有收入、政府有税收**，并使生产、分配、流通、消费循环通畅。
4.  **政府与市场的关系**：构建高水平社会主义市场经济体制，关键是处理好政府和市场的关系，实现**有效市场和有为政府**的更好结合。

### **第七章：社会主义现代化建设的教育、科技、人才战略**
**学习要点：**
1. **教育、科技、人才是全面建设社会主义现代化国家的基础性、战略性支撑**
2. **教育是国之大计、党之大计**
3. **科技自立自强是国家强盛之基、安全之要**
4. **人才是实现民族振兴、赢得国际竞争主动的战略资源**

**小标题：**
* **第一节：全面建设社会主义现代化国家的基础性、战略性支撑**
    1. 坚持教育发展、科技创新、人才培养一体推进
    2. 深入实施科教兴国战略、人才强国战略、创新驱动发展战略
    3. 坚持教育优先发展、科技自立自强、人才引领驱动
* **第二节：加快建设教育强国**
    1. 教育是民族振兴、社会进步的基石
    2. 落实立德树人根本任务
    3. 办好人民满意的教育
* **第三节：加快建设科技强强国**
    1. 科技强则国家强
    2. 打赢关键核心技术攻坚战
    3. 增强自主创新能力
* **第四节：加快建设人才强国**
    1. 培养人才是国家和民族长远发展大计
    2. 培养造就大批德才兼备的高素质人才
    3. 把各方面优秀人才集聚到党和国家事业中来

其余知识点：

1.  **「三个第一」的逻辑**：坚持**科技是第一生产力、人才是第一资源、创新是第一动力**。这三者是全面建设社会主义现代化国家的基础性、战略性支撑。
2.  **教育的根本问题**：必须解决好**培养什么人、怎样培养人、为谁培养人**这一核心课题。立德树人是教育的根本任务，目的是培养社会主义建设者和接班人。
3.  **科技自立自强的紧迫性**：关键核心技术是**要不来、买不来、讨不来的**。核心技术受制于人是最大的隐患，必须发挥**新型举国体制**优势，打赢「卡脖子」技术攻坚战。
4.  **人才强国的标准**：坚持**党管人才**原则，全方位培养、引进、使用人才，重点打造国家战略人才力量。

    3. 积极塑造主流舆论新格局
* **第三节：以社会主义核心价值观引领文化建设**
    1. 广泛践行社会主义核心价值观
    2. 弘扬以伟大建党精神为源头的中国共产党人精神谱系

4.  **中华优秀传统文化的传承**：坚持「**取其精华、去其糟粕**」，推动**创造性转化、创新性发展**（两创）。

### **第十二章：建设社会主义生态文明**
**学习要点：**
1. **绿水青山就是金山银山的科学内涵**
2. **建设美丽中国的主要任务**
3. **全球环境治理的中国方案**

**小标题：**
* **第一节：坚持人与自然和谐共生**
    1. 生态兴则文明兴
    2. 绿水青山就是金山银山
    3. 把生态文明建设摆在全局工作的突出位置
* **第二节：建设美丽中国**
    1. 加快形成绿色生产方式和生活方式
    2. 坚持山水林田湖草沙一体化保护和系统治理
    3. 用最严格制度最严密法治保护生态环境
* **第三节：共谋全球生态文明建设之路**
    1. 保护人类共同家园
    2. 共建清洁美丽世界
    3. 积极推动全球可持续发展
    
其余知识点：

1.  **生态文明的本质观**：**人与自然是生命共同体**。生态兴则文明兴，生态衰则文明衰。
2.  **「绿水青山就是金山银山」的辩证法**：绿水青山既是自然财富，又是社会财富、经济财富。保护生态环境就是**保护生产力**，改善生态环境就是**发展生产力**。
3.  **系统治理的思维**：坚持**山水林田湖草沙一体化保护和系统治理**，不能头痛医头、脚痛医脚。
4.  **「双碳」目标的立场**：实现碳达峰碳中和是我国自主作出的重大决策，路径和节奏必须由**我们自己作主，决不受他人左右**。
